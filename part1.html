Page Designs

For my assignment, I decide to redesign the Lets Talk Science. Because the information is too overwhelming, I think a splash page can filter all the inforamtion for the audience. This is also the reason why I decided to have navigation menu on my splash so it would be convienient for audinence to navigate through the website.

I decide on this desgin for the following reasions regarding to the 4 design principles

  • Proximity I choose to keep the navigation links group together as part of the header which is easy for audience to choose between them
  • Alignment In the header, the logo should aligned to the left and the navigation menu items should algined to the right so it will give the page more of sense of wide spaces. I want to avoid making the webpage being crowded.
  • Repetation I decide to use a title text on the image with the colors that are used for the logo. I also decide to use similar images for the 2 pages.
  • Contrast Using the dark image as background and the light and bright navigation header and footer will draw users attention to the centre of the page.

The Implementation

Splash Page

All the images used for my assignment are the sample photos downloaded from stock.adobe.com

The fonts are downloaded from www.1001freefonts.com

The splash page was bit tricky since I needed to have the image to cover whole page. without leaving any blank. After trying using background image and other settings and research online, I decide to use image with media query. In the end, the result is great with wide page window width.

Get Involved

The content of the page is replica of the original page. credit to http://outreach.letstalkscience.ca/sfu/get-involved.html.

I like the idea of the image background and I decide to use those images as deviders for the content and the images create a flow for the page.

I also ground all the content of the page to the middle so it is easier for reader to focus on the content. I feel that if the content is too spread out , readers might loose their focus.

I also create links to the element on the same page for users' convience. Furthermore, I also create link as block on the image as button to access other parts of website. I decided to play with the blackground color and transparency for user interaction for the link.
